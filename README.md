# Customer Segmentation Project

## Objective
The goal of this project is to segment customers based on their annual income and spending behavior.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- K-Means Clustering

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Feature Scaling
6. K-Means Clustering
7. Customer Segmentation

## Results
The customers were segmented into 5 different groups based on their spending score and annual income.

## Output
Customer segmentation visualization was generated successfully.
